# Vault Canon

`$vault-canon` is the legacy compatibility surface for `$vault-update`.

Follow the `$vault-update` earned-write procedure:

- direct vault writes only for confirmed institutional knowledge;
- routine progress and tentative ideas go to the session doc;
- read the target note before editing;
- preserve frontmatter/provenance and attribution;
- record the write with `$session-update`.

If `$vault-update` is available in context, use it as the preferred generic name. If not, this file is sufficient: do not write directly unless the update is earned and durable.
